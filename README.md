# 低碳算力服务系统智能优化

> 暑期在线自学课程 | 40学时 | 5周完成

在线访问课程主页：**[https://ympianpku.github.io/low-carbon-computing-course/](https://ympianpku.github.io/low-carbon-computing-course/)**

## 课程概述

在国家人工智能发展战略与"双碳"目标深度交汇的时代背景下，构建高效、绿色、可控的算力基础设施已成为关乎国家数字主权与AI产业国际竞争力的核心议题。

本课程设置五个模块，从理论、方法、机制到安全实现全链条覆盖：

1. **分布式智能优化理论** — 异构计算环境下的算力资源协同配置与优化调度
2. **低碳算力服务系统演化规划** — 多源异构数据支撑算力系统布局规划与动态演进
3. **算力-电力协同优化调度** — 多主体参与下算力、能源与碳排放的协同优化
4. **算力服务市场机制设计** — 高效、公平、低碳导向的跨域算力交易与激励机制
5. **低碳算力服务系统动态安全管理** — 数据隐私保护、算法鲁棒性、系统安全与基础设施韧性
